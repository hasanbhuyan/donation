What is the Document Object Model (DOM)?

Document Object Model (DOM) is a platform and language-neutral interface that allows programs and scripts to dynamically access and update the content, structure, and style of a document.

What is event delegation in the contex of the DOM,and why is it useful?

Event Delegation is a pattern based upon the concept of Event Bubbling. It is an event-handling pattern that allows you to handle events at a higher level in the DOM tree other than the level where the event was first received

How do you manipulate an element's attributes and styles using the DOM?

1. Manipulating Attributes

  1. getAttribute(): Retrieves the value of a specified attribute.
  2. setAttribute(): Sets the value of a specified attribute.
  3. removeAttribute(): Removes an attribute from an element.

2. Manipulating Styles

  1. element.style.property: Directly sets an individual CSS property.
  2. element.classList: Adds, removes, or toggles CSS classes for styling.
